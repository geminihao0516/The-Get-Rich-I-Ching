# 使用指南 📖

## 快速入門

### 第一步：準備您的圖片

您需要準備一張包含 **4x4 格線**（共 16 格）的圖片。

#### 圖片要求：
- **格式**：JPG, PNG, WebP, GIF
- **尺寸**：建議至少 800x800 像素
- **大小**：不超過 10MB
- **佈局**：4 行 x 4 列的均勻格線

#### 範例圖片佈局：
```
┌───────┬───────┬───────┬───────┐
│  1格  │  2格  │  3格  │  4格  │
├───────┼───────┼───────┼───────┤
│  5格  │  6格  │  7格  │  8格  │
├───────┼───────┼───────┼───────┤
│  9格  │ 10格  │ 11格  │ 12格  │
├───────┼───────┼───────┼───────┤
│ 13格  │ 14格  │ 15格  │ 16格  │
└───────┴───────┴───────┴───────┘
```

### 第二步：上傳圖片

1. 打開 `index.html` 檔案
2. 點擊上傳區域的「選擇圖片」按鈕
3. 從電腦中選擇您準備好的圖片
4. 等待處理完成（通常只需要 0.5 秒）

### 第三步：開始使用

圖片上傳成功後，您將看到第一張卡片。現在您可以：

#### 🎴 瀏覽卡片

**桌面電腦：**
- 點擊右側箭頭按鈕 → 下一張
- 使用鍵盤方向鍵 `→` → 下一張
- 底部黃色進度條顯示當前位置

**手機/平板：**
- 向左滑動 → 下一張
- 看到提示「向左滑動來財」
- 底部進度條即時更新

#### 💰 洗牌來財

點擊金色的「來財」按鈕：
- 卡片順序會被重新打亂
- 自動跳到第一張
- 顯示錢幣特效動畫
- 伴隨震動動畫效果

#### 🧧 財神爺祝福

點擊左上角的財神爺圖示：
- 財神爺會跳動
- 隨機顯示祝福語：
  - 恭喜發財！
  - 財源廣進！
  - 好運連連！
  - 大吉大利！

#### 🔍 切換視圖

點擊網格圖示按鈕：
- 切換到總覽模式
- 一次查看所有 16 張卡片
- 點擊任意卡片快速跳轉
- 再次點擊切換回單張模式

#### 🗑️ 重置應用

點擊紅色垃圾桶圖示：
- 清除當前圖片
- 回到上傳頁面
- 可以上傳新的圖片

## 進階使用技巧

### 鍵盤快捷鍵

在桌面電腦上使用鍵盤操作更快速：

| 按鍵 | 功能 |
|------|------|
| `→` (右方向鍵) | 下一張卡片 |
| `Space` (空白鍵) | 洗牌來財 |

### 觸控手勢

在觸控裝置上支持流暢的手勢操作：

| 手勢 | 功能 |
|------|------|
| 向左滑動 | 下一張卡片 |
| 點擊卡片 | （無動作，避免誤觸） |

### 計數器說明

畫面左上角的計數器顯示：
```
當前位置 / 總數
例如：5 / 16
```
表示您正在查看第 5 張，共 16 張卡片。

### 進度條

底部的黃色進度條：
- 即時顯示瀏覽進度
- 填滿表示已看完所有卡片
- 洗牌後會重置

## 常見問題解答

### Q: 為什麼圖片上傳後變模糊？
A: 請確保原始圖片解析度足夠高（建議至少 800x800），每格至少 200x200 像素以保證清晰度。

### Q: 可以上傳非 4x4 格線的圖片嗎？
A: 目前版本只支持 4x4 格線（16 格），圖片會被自動平均分割成 4 行 4 列。

### Q: 圖片處理需要多久時間？
A: 通常只需要 0.5 秒左右，具體時間取決於圖片大小和您的裝置性能。

### Q: 我的圖片會被上傳到伺服器嗎？
A: 不會！所有處理完全在您的瀏覽器本地完成，圖片不會離開您的裝置。

### Q: 為什麼手機上只能向左滑動？
A: 為了配合「來財」主題，向左滑動象徵「迎財入室」。如需返回上一張，可使用網格視圖快速跳轉。

### Q: 可以保存當前的卡片順序嗎？
A: 目前版本暫不支持保存功能。關閉頁面後需要重新上傳圖片。建議截圖記錄重要順序。

### Q: 支援哪些瀏覽器？
A: 支援所有現代瀏覽器：
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
- 以及手機瀏覽器

### Q: 可以商業使用嗎？
A: 可以！本專案使用 MIT 授權，可自由使用、修改和商業化。

## 使用建議

### 💡 最佳實踐

1. **圖片準備**
   - 使用高解析度圖片
   - 確保 16 格內容清晰可辨
   - 保持格線均勻對齊

2. **操作技巧**
   - 熟悉鍵盤快捷鍵提升效率
   - 使用網格視圖快速定位
   - 善用洗牌功能增加隨機性

3. **效能優化**
   - 控制圖片檔案大小（2-5MB 最佳）
   - 使用現代瀏覽器
   - 關閉不必要的瀏覽器擴充功能

### ⚠️ 注意事項

1. **圖片內容**
   - 確保內容適合 4x4 平均分割
   - 避免跨格的重要內容
   - 考慮分割線位置

2. **裝置兼容**
   - 舊版瀏覽器可能不支援
   - 低階裝置可能處理較慢
   - 建議使用 WiFi 環境開啟

3. **隱私安全**
   - 不要上傳敏感個人資訊
   - 注意圖片內容的版權
   - 截圖時注意保護隱私

## 錯誤訊息說明

如果遇到錯誤，頁面會顯示友善的提示訊息：

| 錯誤訊息 | 原因 | 解決方法 |
|---------|------|---------|
| 請上傳有效的圖片格式 | 檔案格式不支援 | 使用 JPG/PNG/WebP/GIF 格式 |
| 圖片大小不得超過 10MB | 檔案太大 | 壓縮圖片或調整解析度 |
| 圖片載入失敗，請重試 | 圖片檔案損壞 | 使用其他圖片或重新下載 |
| 檔案讀取失敗，請重試 | 系統讀取錯誤 | 重新選擇檔案 |
| 圖片處理失敗：圖片尺寸太小 | 圖片解析度不足 | 使用更大的圖片 |
